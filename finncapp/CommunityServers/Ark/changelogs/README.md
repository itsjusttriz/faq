# Uploading changelogs

Upload a .diff file. Inside place the additions, changes and removals of each update.
```diff
@@ Added/Updated Mods @@
+ Modname v12345 (link to mod page.)

@@ Removed/Downgraded Mods @@
- Modname v1234.9

@@ Config Changes @@
+ Positive Config change.
- Negative Config change.
```
